# Louzas Framework

Micro framework to serve web applications

This is a very basic framework, made for learning purposes.

In this framework you can create and register routers, retrieve and store information into the database, and modify the controllers.

## Utilization

After downloading this repository into your machine and entering into the folder, initiate a venv:

```shell
python -m venv venv; source venv/bin/activate
```

Install the dependencies:

```shell
pip install -r requirements.txt
```

Run the application:

```shell
python wsgi.py
```
